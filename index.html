PROYECTO CLIENTES

CLIENTES

Este proyecto es una aplicación web que gestiona clientes, permitiendo realizar operaciones como dar de alta nuevos clientes, visualizar un listado de clientes registrados y potencialmente editar o eliminar clientes existentes. La aplicación está construida utilizando Angular, un framework de desarrollo de aplicaciones web en TypeScript.

PROYECTO BLOCKBUSTER CON FIREBASE

BLOCKBUSTER

El proyecto es un sitio web diseñado para la gestión de alquiler de películas. Proporciona a los usuarios una plataforma donde pueden ver información sobre películas disponibles para alquilar, así como también realizar pedidos de alquiler. La interfaz del sitio web incluye secciones dedicadas a mostrar una lista de películas disponibles, junto con sus precios, así como también una sección donde se muestran los detalles de los clientes que han realizado pedidos de alquiler.

PROYECTO LIBROS CON FIREBASE

LIBROS

Esta página muestra un formulario donde puedes ingresar detalles de libros como título, autor, editorial y año de publicación. También hay una lista de libros disponibles en forma de tabla. Cada libro en la lista tiene un botón para seleccionarlo, lo que probablemente permita ver más detalles o editar la información del libro seleccionado.
